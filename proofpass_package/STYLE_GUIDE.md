# ProofPass – Style Guide

## Design Principles
- Minimal
- High contrast
- Data clarity over decoration

## Colors
- Background: #0B0F14
- Primary: #00FFA3 (success)
- Danger: #FF4D4D
- Accent: #5B8CFF

## Typography
- Headings: Bold, tight spacing
- Body: Clean, readable (Inter or similar)
- Labels: Uppercase, small

## Components

### Buttons
- Primary: neon green glow
- Secondary: subtle outline

### Cards
- Dark background
- Soft border
- Slight blur/glass feel

### Status Indicators
- ✅ Green = Verified
- ❌ Red = Failed

## Layout
- Centered flows
- Step-by-step progression
- Avoid clutter

## UX Rules
- One action per screen
- Immediate feedback
- No hidden steps
